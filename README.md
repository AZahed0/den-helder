## ✨ Functionaliteiten

- 🗺️ **Routeplanning**  
  Bereken optimale routes tussen meerdere locaties.

- ⏱️ **Bereikanalyse (Isochronen)**  
  Visualiseer gebieden die bereikbaar zijn binnen een bepaalde reistijd.

- 🔍 **Adreszoeker**  
  Zoek locaties met automatische suggesties (OpenStreetMap).

- 🎨 **Kaartstijlen**  
  Wissel tussen verschillende basemaps.

- 🌙 **Dark mode**  
  Ondersteuning voor licht en donker thema.

- 📦 **GeoJSON export**  
  Exporteer routes en analysegebieden voor gebruik in GIS-software.
